Sentinel Anti-Xray
-------------------
Thank you for installing Sentinel Anti-Xray.

This plugin provides a professional-grade defense against X-ray users
by combining engine-level ray-tracing with heuristic statistics.

Visit our GitHub for full documentation:
https://github.com/Ariqq123/sentinel-antixray

Commands:
/sax gui - Main Dashboard
/sax reload - Reload Configuration
/sax stats <player> - View Mining Stats
